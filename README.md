# Learn Airflow 🚀

Welcome to **Learn Airflow**! This repository is designed to help you quickly set up, learn, and orchestrate automated data pipelines using **Apache Airflow** containerized with **Docker Compose**. 

Whether you are a beginner or looking to experiment with data workflows, this guide will get your local environment running smoothly in just a few steps.

---

## 📋 Prerequisites

Before starting, ensure you have the following installed on your machine:
* **Docker** (Latest version recommended)
* **Docker Compose**
* **Git**

---

Follow these commands in sequence to pull down the project and initialize your multi-container Airflow environment.
